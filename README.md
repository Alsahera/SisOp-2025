a.	348,65410

    348 ÷ 16 = 21 sisa 12 (C)
    21 ÷ 16 = 1 sisa 5

    0,464 × 16 = 7,424 → 7
    0,424 × 16 = 6,784 → 6
    Jadi, 0,654₁₀ ≈ A78

    Hasil dari 348,654₁₀ ≈ 15C,A78
